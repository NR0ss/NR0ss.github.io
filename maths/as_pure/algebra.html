Direct Proof

This requires you to assume that a different statement is true (e.g. 2n is even) and using that, prove that the statement given in the question is true. If asked to prove a number is a multiple of another, you should prove that it can be factorised by that number.

E.g. Prove 2n+2 is a multiple of 2. 2n+2 = 2(n+2). Because 2n+2 can be factorised by 2, it is a multiple of 2. Remember, when ever using direct proof you must write a statement at the end.

Proof by Exhaustion

Proof by exhaustion requires you to prove that the statement given is true for the values given, all this typically requires you to do is substitute the values into the equation.

Disproof by Counter Example

As the name suggests, you are trying to show that the statement is NOT true for all values and this can be done by providing one example where the statement is incorrect. Typically, questions involving this do not offer many marks because it may only require you to do trial and error

What AQA Say You Need To Know:

"Understand and use the structure of mathematical proof, proceeding from given assumptions through a series of logical steps to a conclusion; use methods of proof, including proof by deduction, proof by exhaustion. Disproof by counter example. Proof by contradiction (including proof of the irrationality of √2 and the infinity of primes, and application to unfamiliar proofs)"

Proof by Contradiction is an A Level topic and will be covered later

These questions may include proof by contradiction
